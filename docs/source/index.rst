WEAVE - Workflow Enablement and Advanced Environment
====================================================

WEAVE aims to support user workflows, develop and maintain production workflow tools.
WEAVE is leveraging ICF's idesign to provide a common environment that contains all compatible version of the WEAVE tools and their dependencies.
